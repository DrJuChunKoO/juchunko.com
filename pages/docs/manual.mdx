# 葛如鈞（寶博士）是誰？

葛如鈞（寶博士），現任不分區立法委員，臺灣大學資訊網路與多媒體研究所兼任助理教授。2012年取得臺大資訊工程學博士學位，後赴日本慶応大學媒體設計研究所（KMD）任博士後研究員，曾任日本電通集團台灣 AI/Web3 成長合夥人，也自創知名科技類 Podcast 節目《寶博朋友說》至今不懈。

## 寶博怎麼想

機緣巧合，使命召喚，AI 時代，我們應該致力於嘗試沒做過的事，挑戰大語言模型還沒有建立的事。

### 先進國會，共識未來

「先進」有三層含義：

<Steps>

### 先進科技

第一層含義代表先進科技，「台灣是先進科技之島，但國會、法令、政府單位的效率夠不夠先進呢？」葛如鈞自許把先進科技的概念帶進台灣的政府體制。

### 先進前輩

第二層含義代表他希望能以新身份，向國會先進前輩請教怎麼推動法案。

### 先進國會

第三層則是他先進去國會，大家要隨後來，他認為科技跟生活 很密切，期待4年後會有更多的科技背景的人進入國會。葛如鈞指出，「共識未來」代表看見未來，並凝聚彼此共識。

</Steps>

## 虛擬智慧法規調適

<Callout type="info" emoji="🤓">
  **虛擬智慧法規調適**：國會要先進、AI 要落地、科技做外交。
</Callout>

### 打造台灣成為資通安全安心的科技島嶼

- 修訂「資通安全法」以符合現況
- 落實科技打詐、AI 防詐的政府端責任
- 協助企業增進資訊長、資安長等相關資源與訓練

### 形塑台灣成為虛擬智慧友善的生活環境

- 推動「AI 基本法」三讀通過
- 發展人工智慧、擬人機器人產業
- 透過 AI 人工智慧與虛擬人技術提昇政府便民服務：從 e 化到 v 化

### 促進台灣成為推廣導入 Web3 及元宇宙技術先進之數位社會指標

- 推動「Web3 產業白皮書」並與國際接軌
- 監督數位部數位韌性計畫與相關作為
- 透過立法質詢強化 ACTION, Web3 及元宇宙於產業、教育、文化之連結與導入
