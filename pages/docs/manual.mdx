# 寶博怎麼想

**- 為什麼要做立委？**
🤓：政治不是我的人生目標，但機緣巧合，加上期望在這個 AI 加速進展、元宇宙即將展開的時代，讓台灣政治與法令可以跟上先進科技進展，而非退避三舍待其步步進逼。我們可以更貼近新興科技產業，並在奇點將臨時刻，更快做好準備，我決定掌握機會，貢獻所長，帶來改變。

**- 將來要推動什麼？**
🤓：虛擬智慧法規調適：
因應人工智慧與虛實整合的全境生活型態，建構先進友善的法規環境

一、打造台灣成為資通安全安心的科技島嶼
1. 「資通安全法」
二、形塑台灣成為虛擬智慧友善的生活環境
1. 「人工智慧基本法」
2. 發展人工智慧、擬人機器人產業
3. 便民智慧櫃檯
三、促進 Web3 及元宇宙技術，協助台灣成為先進數位社會指標
1. 「數位中介法」
2. 監督數位部數位韌性計畫與相關作為
3. 國際連結
